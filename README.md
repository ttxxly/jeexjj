#### 项目介绍
jeexjj基于springboot和ssm的极速轻量快速开发框架，能够根据数据库生成单表和一对多表的增删改查代码，使开发节省50%的开发工作量。
使用XJJ可以快速地开发出企业级的Web应用系统。 生成的代码统一规范、统一风格、统一结构便于管理维护。
前端页面使用当前最流行的ACE和bootstrap技术，自适应pc、移动端。
